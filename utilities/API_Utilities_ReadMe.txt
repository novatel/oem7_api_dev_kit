NovAtel OEM7 API Utilities
--------------------------

To make use of the NovAtel API, these utilities are required:
- datablk.exe
- make_iso_hex.bat
- mkisofs.exe
- tosrec.exe

Download these utilites from the NovAtel website here: https://novatel.com/support/firmware-options-pc-software/api

Once downloaded, place these utilities within the OEM7 API folder structure, for example: "\oem7_api_dev_kit\utilities\"
